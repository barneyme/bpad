Help

bEdit.org is a browser based text editor. It can open, create, and save plaintext files like .TXT, .CSV, .XML, .MD, .HTML, .CSS, .JS, .JSON, .SH, .BAT, .PS1 and .PY.

Directions

Open bedit.org in web browser and start writing. Files are automatically saved in your browser. (You should save your file before clearing browser cache however.)

bEdit.org can be used offline after the first use. Reopen bedit.org in your browser to continue working on your file.

Opening bedit.org in another browser tab creates a new file.

↓ displays the current line position.
→ displays the current character position.
= displays the current word count.

Shortcuts

CTRL + N (📄) to create a new file.
CTRL + O (📂) to open a file from your computer.
CTRL + S (💾) to save a file to your computer.
CTRL + F (🔍) to find and replace text.
CTRL + Z (↩) to undo the last change.
CTRL + H (?) to view this help file.

Privacy

bEdit.org does not (and cannot):

  • access your content,
  • criticize your writing or coding,
  • track any activity on your computer,
  • judge your music choices,
  • identify who you are,
  • sell your data.

About

Secondly though I love plain text and I use it every day. Plain text is universal and works on every computer system. Plain text is versatile and can be used for many purposes from scripting to web development to writing. Plain text is lightweight and small in size. Plain text is human readable. Plain text is distraction free because you just have the text to concentrate on.

Lastly bEdit can be used on any computer without having to install anything and with a consistent user interface.
