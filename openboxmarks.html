Import Weblinks

Help - Import Format

File Format

Create a text file (.txt) with your weblinks organized by folders. Use the following format: 

[Folder Name]
Link Title = https://example.com
Another Link = https://another-site.com

[Development]
GitHub = https://github.com
Stack Overflow = https://stackoverflow.com
MDN Docs = https://developer.mozilla.org

[News]
BBC News = https://bbc.com
Reuters = https://reuters.com

Format Rules

  • Folder names are enclosed in square brackets: [Folder Name]
  • Links follow the format: Link Title = URL
  • URLs must include http:// or https://
  • Empty lines are ignored
  • Lines starting with # are treated as comments and ignored

Example Structure

  • Each folder section starts with a name in brackets
  • All links after a folder heading belong to that folder
  • You can have multiple folders in one file
  • Existing folders will be merged with imported ones
